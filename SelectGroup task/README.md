# Autogen SelectGroup 实验作业

## 作业内容

- 课堂任务1：SelectorGroup基础实验
- 课堂任务2：状态持久化与加载
- 课后任务1：SelectGroupTeam分工协作报告

## 运行方法

1. 安装依赖
   ```bash
   pip install pyautogen
   ```

2. 运行各任务脚本

   - 课堂任务1
     ```bash
     python selectgroup_demo.py
     ```
   - 课堂任务2
     ```bash
     python save_state.py
     # 再运行
     python load_state.py
     ```
   - 课后任务1
     ```bash
     python report_team.py
     ```
